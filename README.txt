== Description ==

WP Chat AI is a WordPress plugin that allows visitors to chat with your website using an AI-powered chatbot. The plugin integrates with OpenAI's GPT (Generative Pre-trained Transformer) model to provide intelligent responses to user queries.

Features:
- Chat with visitors on your website in real-time.
- AI-powered responses based on user input.
- Easy-to-use admin panel to configure plugin settings.

== Installation ==

1. Upload the `wp-chat-ai` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the WP Chat AI settings page to configure the plugin.

== Usage ==

Once the plugin is activated and configured, a chat popup will appear on your website allowing visitors to chat with the AI chatbot.

== Frequently Asked Questions ==

= How do I configure the plugin? =

After activating the plugin, go to the WP Chat AI settings page in the WordPress admin panel. Here, you can set up your OpenAI API key and enable/disable the chat feature.
